Introduction

This is a living document that represents the guidelines for each individual UI component in the Boyne app.

Usage

For each new component created, we will need to copy the markup found in the individual Razor templates/cshtml files:

All Razor Templates can be found in the following directory

byn.website/Views/Shared/

Example: Header.cshtml

Original code from byn.website/Views/Shared/Layout/_Header.cshtml


@model Ingeniux.Runtime.ICMSPage
@using Ingeniux.Runtime.Library.Models.Pages
@{
    var page = new PageView();
    page.Map(Model);
    //  var page = ModelFactory.GetModel<PageView>(Model);
    <ul>
    @foreach (var primaryNavItem in page.SiteControl.Navigation.Navigations)
    {
        <li>@primaryNavItem.Name</li>
    }
    </ul>
}

We want to take the original HTML and convert it into markup for the style guide.

Output:

    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
    </ul>

With each output, we should be able to copy and paste the html into new *.cshtml files to create new components.
